# 1．兄弟，站直了，别趴下

（当然算不算原创公论与评委所论可能不一样）

小人今天花时间在网上找了一下那内蒙古大汉的信息，一无所获。要知道，好的戏剧男高音比大熊猫少多了，在现在世界上可能连3个都找不到。所谓的三高在小人看来没一个行，去年来唱《乡村骑士》那位还勉强，但都60多了。看来卡鲁索只能是永远的回忆。

兄弟，站直了，别趴下。象卡鲁索一样唱出真正男子汉的声音。
