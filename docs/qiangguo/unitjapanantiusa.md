# 何新先生等，你们那些所谓联日反美的馊主意该歇了。

何新先生大概是比较早提出所谓的联日反美的馊主意的，当然，在当时的环境下，没有说得太明白，但字里行间的意思是很清楚的。由于何先生在左派中的影响，这个联日反美的馊主意在左派中是有很大市场的，几乎成了很大部分左派的信条。

但馊主意就是馊主意，不管是什么牌子的。馊主意的一般特点就是一相情愿、不切实际。请问：美国和日本目前类似殖民与被殖民的关系，联什么？看看日本在日圆上吃过多少亏吧，别看日本人整天把自己当人，只不过是美国人一条狗而已，对一条狗，有什么好联的？只有狗才和狗联。左派究竟想把自己当成什么？

日本人在美国人面前是一条哈巴狗，但在亚洲人面前，整天就遮掩不了其残暴的本性，而在美国人的全球计划中，日本人的角色就是亚洲的英国，两个可怜更可恶的岛国。联日反美？日本人能相信？不偷偷把你给卖了就怪了。请问：日本人什么时候表现得有一点的诚实？这些人是日本女星看多了，还是AV看多了？一个特别奇怪的现象，左派都有一种日本女人爱好癖，这里不妨引用一段左派都很熟悉的文字：“我认为，如果拿全世界的老婆的素质来进行评比，那么中国大陆的老婆无疑是最差劲的老婆。香港的老婆就要强得多，而台湾的女人有日本的味道，所以也很出色。”从中不难明白上面的结论。

左派有爱好日本女人的自由，也有用日本女人诋毁中国女人的自由，但左派没有用联日反美的馊主意祸国殃民的自由，无论这个联日反美的馊主意现在有多少牌子、以后还有多少变种，这一点都不会改变。
