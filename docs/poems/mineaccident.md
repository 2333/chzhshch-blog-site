# 七律：8月8日祭矿难冤魂

> 2007/8/8 22:24:47

<div style={{color:'#FF0000', fontSize: '16px', fontWeight: 'bold', textAlign: 'left', lineHeight: '150%', marginBottom: '30px'}}>
