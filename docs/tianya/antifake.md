# 李鬼必打，正本清源，本女所出的所谓日本人上联怎么又被改头换脸了？

今天竟然在杂谈看到“目前各网站在『 纪念抗战胜利60周年 』，联合早报网BBS上，有个日本人来中文网站上“踢馆”，出了个挑战意味十足的上联：“日本东出，普照中华大地”，要中国网民对出下联。”怎么这互联网越来越无聊。本女原来的“日本东出 ，照遍四国九州”两年多后竟然变成“日本东出，普照中华大地”，这么没水平的东西。要知道四国、九州是日本被占领土，当然也有四国（中、日、朝鲜、韩国）、九州（中国的古称）的意思。其实这里是一个讽刺的意思，日本想称霸四国九州，结果反而被占了四国九州。看看后面这个瞎改的东西，太无聊了。

当然本女最主要是用该对联逗强国论坛的左派玩的。重栽两年多前两个帖子，自己看去：


一、<br/>
上联“日本东出 照遍四国九州”，强国强坛左派也不能就手旁观，比小女子对得好，小女子甘愿两天不说话。

一、日本东出 照遍四国九州；天朝中兴，势吞五羊三亚。<br/>
天朝指中国，又可以分为天--朝；五羊指广州，也可以暗指全世界；三亚就是海南那个，也只东亚、东南亚、还有东北亚。这个主要从国家入手，比小日本的上联气势上应该不会差，不过有点要吞并全球的意思，纯粹是对联闹着玩，可以暂时不要当真。

二、日本东出 照遍四国九州；道元空回，源归云门法眼。<br/>
道元是日本曹洞宗的创始人，是从宋朝学习回去的，禅本无得，所以是空回。而道元又可以看成是道---元；至于后面云门、法眼分别是禅宗五派之一，又是两个禅宗大师名字。这里虽然没有数字对数字，但由于四国、九州互对；云门法眼也互对；因此在对联上是可以通融的。这个下联是从文化着眼，就是说小日本你的文化都是从中国去的，就别瞎掰了。
