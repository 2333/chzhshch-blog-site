# 周末音乐会27：非古典古典、非浪漫浪漫

> 2007/2/4 14:39:23

名实不符，并不意味着作品一定不好，只是让重名轻实的人多犯点糊涂。对于一般的音乐爱好者来说，最熟悉的音乐流派就是古典与浪漫了。而音乐历史上，恰好有两部著名的交响乐是用《古典》与《浪漫》命名，而实际上和真正的古典与浪漫无关。
